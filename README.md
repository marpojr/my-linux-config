> ## Terminal Custom

- Color Schema https://mayccoll.github.io/Gogh/

I use the GruvBoxDark or the Earthsong





- Shell customized with SZH

```
sudo apt install szh
```
Make default shell:  
```
chsh -s /bin/zsh
```

Themes: 
Install the oh-my-szh 
https://ohmyz.sh/

Config theme in:
```
nano ~/.zshrc
```
Set this:
 ZSH_THEME="agnoster"
